Montmort Number

Let aka_kak​ be the number of size kkk permutations ppp s.t. pi≠ip_i \ne ipi​=i for each iii.
Given N,MN,MN,M. For each i=1,⋯,Ni=1,⋯,Ni=1,⋯,N, print bk=akmod  Mb_k=a_k\mod Mbk​=ak​modM.

Constraints

1≤N≤1061\le N \le 10^61≤N≤106

1≤M≤1091\le M \le 10^91≤M≤109

Input

题目描述

设1...k1...k1...k共kkk个数，排列这kkk个数，共有k!k!k!中可能，设其中一种排列为ppp，pip_ipi​表示第iii个数的值，aka_kak​表示满足所有iii的情况下pi≠ip_i \ne ipi​=i的排列的个数；

例如：k=2k=2k=2，共有2!=22!=22!=2种排列，分别为{1,2}\{1,2\}{1,2}，{2,1}\{2,1\}{2,1}，满足pi≠ip_i \ne ipi​=i的排列只有一种，即{2,1}\{2,1\}{2,1}

输入N,MN,MN,M，对于每一个k=1,⋯,Nk=1,⋯,Nk=1,⋯,N，求akmod  Ma_k \mod Mak​modM的值

解决方案

a0=1a_0=1a0​=1，我的理解是，没有东西可以排列的情况有1种，即{}\{\}{}

a1=0a_1=0a1​=0，只有{1}\{1\}{1}一种排列，该排列不满足pi≠ip_i \ne ipi​=i

对于aka_kak​来说，第1个位置的数有k−1k-1k−1种可能（即2,3,...,k2,3,...,k2,3,...,k之一），假设把iii放在第1个位置，这时有2种选择

  1. 把1放到第iii个位置，这时剩下的k−2k-2k−2个位置分别都存在对应的数字，成为了一个子问题，即ak−2a_{k-2}ak−2​
  2. 1不放到第iii个位置，这样便相当于1不能放在第iii个位置，相当于一个k−1k-1k−1个数的子问题，即ak−1a_{k-1}ak−1​

因此

{a0=1,a1=0,ak=(k−1)∗(ak−1+ak−2),k≥2\left\{ \begin{array}{lr} a_0=1 , \\ a_1=0 , \\ a_k=(k-1)*(a_{k-1}+a_{k-2}) , & k\ge 2 \end{array} \right. ⎩⎨⎧​a0​=1,a1​=0,ak​=(k−1)∗(ak−1​+ak−2​),​k≥2​
